# 卡牌项目客户端

### 编译配置

| 环境        | 说明           |
|------------|--------------|
| Node       | v16.14.0     |
| 开发工具     | vscode或其他熟悉的 |

### 编译调试
1. 先安装pnpm包`npm install -g pnpm`
2. 安装依赖`pnpm install`
3. 本地调试`pnpm dev`或`npm run dev`
4. 打包编译`pnpm build` 或`npm run build`

### 客户端配置
修改配置中的接口地址
- 测试环境配置`.env.dev`
- 正式环境配置`.env.prod`
